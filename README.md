# Library Management System (C++)

A console-based **Library Management System** developed in **C++** using **Object-Oriented Programming (OOP)** and **File Handling**. This application allows users to manage library books by adding, searching, issuing, returning, and displaying book records through a menu-driven interface.

---

## Features

- Add New Book
- Display Book Details
- Search Book by Title
- Issue Book
- Return Book
- Store Book Records Using File Handling
- Menu-Driven Console Interface

---

## Technologies Used

- C++
- Object-Oriented Programming (OOP)
- File Handling (`fstream`)
- Functions
- Classes & Objects

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/Library-Management-System-CPP.git
   ```

2. Open the project in any C++ IDE such as:
   - Visual Studio Code
   - CodeBlocks
   - Dev-C++
   - Code::Blocks

3. Compile the program:
   ```bash
   g++ main.cpp -o LibraryManagement
   ```

4. Run the executable:
   ```bash
   ./LibraryManagement
   ```
